# 화면별 텍스트 와이어프레임

## 1. 시작 화면

```text
+--------------------------------------------------+
|                 도레미 마스터                    |
|      악보를 보고 건반을 눌러보자                 |
|                                                  |
|          [ 닉네임 입력________________ ]          |
|                                                  |
|              [ 시작하기 ] [ 이어하기 ]            |
|                                                  |
|                    [ 설정 ]                      |
+--------------------------------------------------+
```

## 2. 레벨 선택 화면

```text
+--------------------------------------------------+
| 닉네임: 민준                                     |
|                                                  |
| [Level 1] 높은자리표 기본 음계                   |
| [Level 2] 낮은자리표 기본 음계 (잠금)            |
| [Level 3] 덧줄 음표 (잠금)                       |
| [Level 4] 샵/플랫 (잠금)                         |
|                                                  |
| 문제 수: (10) (20) (무한)                        |
| 힌트 보기: [ON/OFF]                              |
|                                                  |
|              [ 학습 시작 ]                       |
+--------------------------------------------------+
```

## 3. 퀴즈 화면

```text
+--------------------------------------------------+
| Level 1      문제 3/10      점수 20              |
|                                                  |
|                 높은자리표                        |
|                                                  |
|              ----- 음표 1개 표시 -----           |
|                                                  |
|             정답/오답 피드백 영역                |
|                                                  |
|   [도] [레] [미] [파] [솔] [라] [시] [도]        |
|                                                  |
|                가상 피아노 건반                  |
+--------------------------------------------------+
```

## 4. 결과 화면

```text
+--------------------------------------------------+
|                 학습 완료                         |
|                                                  |
|                총 점수: 80점                      |
|                정답: 8 / 10                       |
|                정답률: 80%                        |
|                                                  |
|      가장 많이 틀린 음: 시(B4)                    |
|                                                  |
|         [ 다시하기 ] [ 홈으로 ]                  |
+--------------------------------------------------+
```

## 5. UI 원칙

- 한 화면에 핵심 행동은 하나만 둔다.
- 텍스트보다 큰 버튼과 큰 음표를 우선한다.
- 피드백 문구는 짧고 긍정적으로 유지한다.
