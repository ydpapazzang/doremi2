# 초기 개발 순서

## 1. 프론트엔드

1. React + Vite + TypeScript 초기화
2. 기본 라우팅 추가
3. 시작 화면과 레벨 선택 화면 구현
4. 퀴즈 화면 레이아웃 구현

## 2. 백엔드

1. FastAPI 앱 생성
2. SQLite 연결
3. `users`, `learning_sessions`, `question_logs`, `level_configs` 모델 작성
4. 사용자/세션 API 생성

## 3. 첫 통합 목표

1. 닉네임 입력
2. 세션 시작 API 호출
3. 퀴즈 화면 진입
4. 10문제 세션 종료 후 결과 표시

## 4. 첫 번째 데모 완료 기준

- Level 1만으로 10문제 플레이 가능
- 피아노 음 재생 가능
- 결과 저장 가능
